# AI & ML Assignments

This repository contains completed assignments 1 to 5 for the AI & ML course.

## Folder Structure

Each assignment folder contains the following:

- **Source Code:** Python notebooks or scripts implementing the assignment tasks.
- **Dataset:** Data files used for the assignment.
- **Outputs:** Results or outputs generated from the code (such as result files, images, or reports).
- **Assignment Questions:** The text or PDF of the assignment instructions or questions.

The structure is as follows:

- `Assignment1/`
  - Source code
  - Dataset
  - Assignment questions
- `Assignment2/`
  - Source code
  - Dataset
  - Assignment questions
- `Assignment3/`
  - Source code
  - Dataset
  - Assignment questions
- `Assignment4/`
  - Source code
  - Dataset
  - Assignment questions
- `Assignment5/`
  - Source code
  - Dataset
  - Assignment questions
